<div className="mt-16 mb-16 max-w-5xl mx-auto px-6">
  <h2 className="text-3xl font-bold text-gray-900 dark:text-white mb-4">Quick start</h2>
  <p className="text-lg text-gray-600 dark:text-gray-400 mb-8">Get your RAG system running in minutes</p>
  
  <Steps>
    <Step title="Configure Azure services">
      Set up Azure AI Search and Azure OpenAI in your Azure portal. Create a search index and deploy a chat model.
      
      <Accordion title="View configuration example">
        ```json appsettings.json
        {
          "AzureOpenAI": {
            "Endpoint": "https://<your-openai>.openai.azure.com/",
            "ApiKey": "<your-api-key>",
            "ChatDeployment": "chat",
            "ApiVersion": "2024-10-21"
          },
          "AzureAISearch": {
            "Endpoint": "https://<your-search>.search.windows.net",
            "ApiKey": "<your-api-key>",
            "IndexName": "rm-rag-index"
          }
        }
        ```
      </Accordion>
    </Step>
    
    <Step title="Clone and configure">
      Clone the repository and update your `appsettings.json` with Azure credentials.
      
      ```bash
      git clone https://github.com/robertomenciasoftwareone/real-madrid-rag.git
      cd real-madrid-rag/src/RagConsole
      # Edit appsettings.json with your Azure credentials
      ```
    </Step>
    
    <Step title="Run the console">
      Build and run the RAG console to start asking questions.
      
      ```bash
      dotnet run
      ```
      
      <Note>The console validates your configuration on startup and prompts for questions interactively.</Note>
    </Step>
  </Steps>
</div>
