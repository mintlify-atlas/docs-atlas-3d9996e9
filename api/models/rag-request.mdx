## Overview

The `RagRequest` model encapsulates a user's question that will be processed through the RAG pipeline. It serves as the input to the orchestrator, triggering both the Azure AI Search retrieval and Azure OpenAI answer generation.

## Definition

```csharp
public sealed record RagRequest(string Question);
```

## Properties

<ParamField path="Question" type="string" required>
  The user's question or query to be answered using RAG. This question will be used to:
  - Search for relevant documents in Azure AI Search
  - Generate a contextualized answer using Azure OpenAI
</ParamField>

<Note>
  `RagRequest` is implemented as a C# record type, providing immutability and value-based equality by default.
</Note>

## Usage Example

The following example shows how `RagRequest` is used in the RAG orchestrator:

```csharp
// From RagOrchestrator.cs:22
public async Task<RagResponse> AskAsync(string question, CancellationToken cancellationToken = default)
{
    var request = new RagRequest(question);
    var sources = await azureSearchService.SearchAsync(request.Question, cancellationToken);
    var answer = await azureOpenAiService.GenerateAnswerAsync(request.Question, sources, options.SystemPrompt, cancellationToken);
    return new RagResponse(answer, sources);
}
```
